BDFD utility top.gg

Voter Only Commands.

Prompt users to vote on Top.gg, then claim to unlock exclusive commands. Verified via the Top.gg API. Access expires after 12 hours and resets automatically.

// how to use

Add all commands with their triggers. Get your API key from Top.gg. Create user variables voter (value: false) and voter-timestamp (value: empty).

// credits

Inspired by a random idea. Uses the Top.gg REST API to verify votes. The 12-hour cooldown matches Top.gg's own voting cooldown window. Includes a timestamp check for auto-expiry.
