# Zubariel.

This repository hosts the source code for my personal website: [https://zubariel.is-a.dev/](https://zubariel.is-a.dev/).  

It’s a multi-page, static website designed to showcase my portfolio, projects, and personal links in a clean, modern layout.

---

## About the Website

The site serves as a centralized hub for my work and online presence. Features include:

- A landing page with quick access to my projects, portfolio, and external profiles.  
- Dedicated pages highlighting coding projects, portfolio work, and reusable code snippets.  
- Contact functionality to reach out directly.  
- Real-time status updates and integrations with platforms like Anilist and MangaList.  

Everything is built with **HTML, CSS, Tailwind CSS, and JavaScript**, keeping the site fast and lightweight.
